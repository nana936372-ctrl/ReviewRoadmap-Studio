# Product Workbench Redesign Plan

## Goal

Turn the demo from a report-like page into a simple decision workbench that shows business value quickly, keeps score explanation behind click-through detail, and makes the one-page brief feel portfolio-ready.

## Changes

1. Add a compact top decision summary in the first viewport.
2. Compress the sample analysis controls into a setup bar with advanced options tucked away.
3. Replace the sentiment table with a visual signal map and sentiment bars.
4. Make roadmap cards read as concise action cards while keeping score buttons as detail entry points.
5. Add factor contribution bars to the score detail page.
6. Reframe the AI workflow as a compact stepper.
7. Strengthen the decision brief as an executive memo.
8. Simplify colors, spacing, borders, and responsive behavior.

## Verification

- Update tests for the new information architecture before implementation.
- Run the component/app test suite.
- Verify the running page in browser at desktop and mobile widths.
